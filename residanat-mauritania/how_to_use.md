# Comment Utiliser l'Application

## 🎯 Étapes pour Utiliser les Boutons

### ⚠️ IMPORTANT: Sélectionner un Objectif d'Abord!

Les boutons **"Lire le Cours"** et **"S'entraîner"** nécessitent que vous sélectionniez d'abord un objectif dans le menu.

### 📋 Procédure Correcte:

1. **Ouvrir le Menu (si sur mobile)**
   - Cliquez sur l'icône ☰ en haut à gauche

2. **Sélectionner un Objectif**
   - Dans le menu de gauche, choisissez une catégorie:
     - Sciences Fondamentales
     - Pathologies Médicales
     - Pathologies Chirurgicales
   - Cliquez sur un objectif spécifique (ex: "Électrophysiologie cardiaque")

3. **Utiliser les Boutons**
   - Une fois l'objectif sélectionné, les boutons deviennent actifs:
     - 📖 **Lire le Cours** - Ouvre le PDF
     - 🧠 **S'entraîner** - Lance les quiz
     - 📄 **Imprimer les QCMs** - Génère un PDF
     - 📊 **Statistiques** - Affiche vos progrès

## 🔴 Message d'Erreur

Si vous cliquez sur un bouton sans avoir sélectionné d'objectif, vous verrez:

```
⚠️ Veuillez d'abord sélectionner un objectif dans le menu de gauche.
```

**Solution:** Suivez les étapes ci-dessus pour sélectionner un objectif.

## 📱 Sur Mobile

- Le menu s'ouvre automatiquement si vous essayez d'utiliser un bouton sans objectif sélectionné
- Sélectionnez votre objectif
- Le menu se ferme automatiquement après la sélection

## 💻 Sur Desktop

- Le menu est toujours visible à gauche
- Cliquez simplement sur un objectif
- Les boutons deviennent immédiatement utilisables

## ✅ Indicateur Visuel

Quand un objectif est sélectionné:
- ✅ Le titre de l'objectif apparaît en haut de la page
- ✅ L'objectif est surligné dans le menu
- ✅ Les boutons d'action sont visibles

## 🎓 Exemple Complet

### Scénario: Je veux réviser l'Électrophysiologie cardiaque

1. **Ouvrir le menu** (si mobile)
2. **Cliquer sur** "Sciences Fondamentales"
3. **Cliquer sur** "Électrophysiologie cardiaque"
4. **Voir** le titre "Objectif: Électrophysiologie cardiaque"
5. **Cliquer sur** "Lire le Cours" ou "S'entraîner"
6. **Profiter** de votre session d'étude!

## 🔧 Dépannage

### Les boutons ne répondent toujours pas?

1. **Vérifiez** qu'un objectif est sélectionné (titre visible en haut)
2. **Rafraîchissez** la page (F5)
3. **Vérifiez** votre connexion internet
4. **Ouvrez** la console du navigateur (F12) pour voir les erreurs

### Le menu ne s'affiche pas sur desktop?

1. **Vérifiez** la largeur de votre fenêtre (>768px)
2. **Rafraîchissez** la page
3. **Désactivez** les extensions de navigateur qui pourraient interférer

## 📞 Support

Si le problème persiste:
- Vérifiez que tous les fichiers sont présents
- Assurez-vous que le serveur web fonctionne
- Consultez les logs de la console (F12)
